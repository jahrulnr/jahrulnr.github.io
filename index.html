Tentangku

Saya memiliki 3 tahun pengalaman sebagai developer dengan fokus pada Golang, PHP (Laravel), AWS, manajemen server Linux/on‑premise, dan pipeline CI/CD menggunakan GitLab. Di pekerjaan sehari-hari saya merancang dan membangun secure APIs serta microservices yang scalable — salah satu contohnya adalah backend untuk Telkomsel - OMNI, di mana tim kami menerapkan otentikasi, rate‑limiting, dan monitoring untuk menjaga keamanan dan ketersediaan layanan.

Selain backend, saya juga mengembangkan aplikasi Android (Java) dan melakukan prototyping embedded (ESP32). Pendekatan saya pragmatis: rapid prototyping, real-device testing, lalu iterasi berdasarkan metrics dan user feedback. Kadang-kadang, saya menulis artikel teknis untuk mendokumentasikan solusi dan berbagi lessons‑learned; contoh tulisan dapat dibaca di Medium dan BangunSoft.
